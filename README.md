## Calculadora de Matrices
-------

# Caracteristicas 
- Entrada y Salida
- Operaciones con Matrices:
  Suma, Resta, Multiplicacion por escalar, Multilpicacion de matrices
  Matriz inversa y Traspuesta
- Solucion de sistemas de ecuaciones lineales con matriz inversa 

## Como poder utilizar el programa
-----
1. Descargar main.py
2. Descargar la carpeta de entradas con sus respectivos archivos
3. Descargar la carpeta de matrices con sus respectivos archivos 
4. Colocar todo en una misma carpeta la cual incluya: main.py, entradas, matrices
5. Abrir la carpeta en un editor de codigo
6. Ejecutar main.py
